## Document Contract
每份文件皆包含 Purpose、Scope、Principles、Standards、Rules、Validation、Related Documents 與 Version。規範衝突時，以治理文件、設計 Token 與版本較新的正式規範為準。

## Change Control
所有重大變更必須同步更新版本、交叉引用、品質檢查與 `CHANGELOG.md`。禁止只修改視覺結果而未更新來源規範。
