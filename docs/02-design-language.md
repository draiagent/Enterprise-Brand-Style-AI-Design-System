# 02 — Design Language

## Purpose
將品牌哲學轉化為可重複的視覺語法與互動表現。

## Scope
涵蓋形狀、層級、密度、節奏、深度、動態與跨媒體一致性。

## Principles
- 專業但不僵硬。
- 科技但不冷漠。
- 高資訊密度但保持清楚。
- 活潑色彩服務於分類，不作無目的裝飾。
- 重要訊息以層級、空間和對比表達。

## Standards
### Shape
主要容器採一致圓角級距；資訊卡、標籤、提示與按鈕不得任意使用不同圓角。

### Depth
陰影僅表示層級或可操作性。禁止多重強烈陰影、玻璃效果與發光同時堆疊。

### Density
企業報告與儀表板採中高密度；招生海報與社群素材採中密度；封面與主視覺可降低密度但不得失去必要資訊。

### Hierarchy
每一畫面只設一個第一視覺焦點，最多兩個次焦點。標題、摘要、證據、行動按鈕依序排列。

## Rules
- 不混用三種以上視覺風格。
- 不以背景特效降低文字可讀性。
- 同一系列輸出必須共享 Token、元件與圖像語言。
- 動態效果應解釋狀態，不作無目的循環動畫。

## Validation
檢查焦點、層級、風格數量、元件一致性與跨媒體辨識度。
