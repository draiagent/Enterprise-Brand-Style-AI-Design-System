# 08 — Illustration System

## Purpose
定義品牌插畫、角色、場景與概念圖的共同視覺語言。

## Scope
適用於品牌人物、科技場景、流程概念、教育教材、社群素材、封面與主視覺。

## Principles
插畫必須支援訊息與品牌辨識；角色外觀、比例、服裝、道具與色彩需可持續重現。

## Standards
### Style Bible
每套插畫需記錄：角色設定、身形比例、臉部特徵、主要服裝、品牌色、材質、光線、鏡頭、背景複雜度與禁止元素。

### Composition
主體應具清楚輪廓，背景不得與文字競爭；留白應依後續排版位置預留，而不是生成後勉強覆蓋。

### Continuity
同一人物跨素材使用相同角色參考與版本識別。角色重大改版需經治理程序，不得由單次 Prompt 任意改變。

## Rules
- 不使用未授權真人肖像或混淆品牌角色的近似形象。
- 不讓 AI 自行生成 Logo 與重要品牌文字。
- 插畫不得造成產品功效、數據或專業事實誤導。
- 所有正式插畫應登錄於 `assets/`。

## Validation
檢查角色一致性、品牌資產正確性、文字留白、內容適切性及來源紀錄。
