# Assets

## Purpose
正式品牌資產（Logo、核准角色插畫、核准圖片、字體授權檔）的登錄處。依 `../docs/01-brand-guide.md`：品牌人物與 Logo 必須由這裡的登錄來源取得，不得用截圖或未核准版本替代。

## Status
**目前登錄為空。** 這個 repo 手動建立的階段還沒有任何正式核准的品牌資產——沒有 Logo 檔、沒有核准的角色插畫、沒有字體授權檔。這不是遺漏，而是如實反映現況：在 Owner 提供或核准實際資產之前，不應該虛構或假設任何資產已經存在。

## Registry Contract
每個資產登錄至少要有：

| 欄位 | 說明 |
|---|---|
| Name | 資產名稱 |
| Owner | 核准者／擁有者 |
| Version | 版本 |
| License / Usage Status | 授權或使用狀態（例如：內部使用、公開使用、有效期限） |
| File | 實際檔案路徑 |
| Approved By | 核准人與核准日期 |

## Registry

_（目前無登錄項目）_

## Rules
- 禁止用截圖取代正式 Logo 檔案（見 `../docs/01-brand-guide.md`）。
- 禁止在沒有登錄的情況下，讓 AI 生成的角色／Logo 被當作正式資產重複使用。
- AI 生成的插畫若要成為可重用的正式角色，需先走過 `../docs/14-brand-governance.md` 的核准流程，登錄進這裡才算數。
