# Checklist — Infographic

先過 `quality-gate.md`，再檢查以下項目（彙整自 `templates/infographic.template.md`、`docs/05`、`docs/07`）。

- [ ] 標題與一句話結論清楚呈現
- [ ] 3–7 個主要資訊群組，每個區塊只表達一個主要觀點
- [ ] 數字、單位與來源一致，且可追溯
- [ ] 圖示含義一致，符合 `docs/07-icon-system.md`
- [ ] 沒有用裝飾取代資訊（例如純美觀但無意義的圖形）
- [ ] 版面在目標比例（1:1／3:4／4:5／9:16）下，標題、核心訊息、CTA 都保留在安全區內
- [ ] 若是系列 Carousel，每張都使用相同的 `tokens/` 與 `components/`，風格一致
- [ ] 編號／步驟標記（如有）對應真實的順序關係，不是為了裝飾而加
