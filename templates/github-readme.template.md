# GitHub README Template

## Metadata
- Repository:
- Owner:
- Version:
- License:
- Status:

## Purpose
建立清楚說明定位、安裝、結構、使用與治理方式的 GitHub 專案首頁。

## Inputs
- 專案名稱：
- 一句話描述：
- 目標使用者：
- 核心功能：
- 安裝方式：

## Outputs
- `README.md`
- 導覽連結
- 使用範例

## Required Sections
1. Project Overview
2. Key Capabilities
3. Repository Structure
4. Quick Start
5. Usage
6. Documentation
7. Governance and Security
8. Contributing
9. License and Version

## Optional Sections
- Badges
- Screenshots
- Roadmap
- FAQ

## Validation Rules
- 所有相對路徑可開啟。
- Quick Start 可由新使用者執行。
- 不放置密碼、Token 或內部敏感資訊。
- 版本與 `VERSION`、`CHANGELOG.md` 一致。

## Example
```markdown
# Project Name
一句話描述。

## Quick Start
1. Clone repository
2. Configure environment
3. Run validation
```
