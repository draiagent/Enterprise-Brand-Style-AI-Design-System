## Required Sections
1. Cover
2. Context and Problem
3. Insight
4. Framework or Solution
5. Evidence
6. Implementation
7. Risks
8. Call to Action

## Optional Sections
- Agenda
- Case Study
- Appendix
- Q&A

## Validation Rules
- 每頁只保留一個主要訊息。
- 標題應表達結論，不只描述主題。
- 圖表需標示來源、期間與單位。
- 遵循 `../docs/12-presentation-standard.md`。
