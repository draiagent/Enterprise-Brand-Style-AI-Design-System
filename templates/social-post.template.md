# Social Post Template

## Metadata
- Campaign:
- Platform:
- Owner:
- Version: 1.0.0
- Publish Date:

## Purpose
建立符合品牌語氣、平台限制與行動目標的社群貼文。

## Inputs
- 溝通目標：
- 目標受眾：
- 核心訊息：
- 素材：
- CTA：
- 禁止聲明：

## Outputs
- 主貼文
- 短版文案
- 標題與關鍵字
- 圖像規格

## Required Sections
1. Hook
2. Problem or Context
3. Value or Insight
4. Evidence
5. CTA
6. Hashtags or Keywords

## Optional Sections
- 引言
- 案例
- 活動資訊
- 法規聲明

## Validation Rules
- 前兩行可獨立傳達重點。
- 文案不可誇大、虛構或加入未核准功效。
- 平台字數、圖片比例與連結規則正確。
- 品牌名稱、人物職稱與日期經核對。

## Example
```yaml
platform: Facebook
length: 200 字
hook: AI 不只是工具，而是企業作業系統
cta: 查看完整架構
```
