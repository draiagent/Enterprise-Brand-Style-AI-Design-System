# Markdown Note Template

## Metadata
- Title:
- Type:
- Tags:
- Created:
- Updated:
- Version: 1.0.0
- Status:

## Purpose
建立可放入 Obsidian、GitHub 或企業知識庫的結構化筆記。

## Inputs
- 主題：
- 原始資料：
- 使用情境：
- 相關專案：

## Outputs
- Markdown 筆記
- 來源與交叉引用
- 待辦或決策紀錄

## Required Sections
1. 摘要
2. 核心概念
3. 架構或流程
4. 實務應用
5. 風險與限制
6. 行動清單
7. 相關筆記與來源

## Optional Sections
- Mermaid 圖
- Dataview 欄位
- Q&A
- 版本差異

## Validation Rules
- 刪除重複內容並保留來源。
- 標題層級連續且連結有效。
- 事實、推論與建議分開標示。
- 可使用相對路徑連結其他文件。

## Example
```yaml
title: 企業 AI 導入八層架構
tags: [AI, Enterprise, Architecture]
status: reviewed
```
