# Poster Template

## Metadata
- Title:
- Project:
- Owner:
- Version: 1.0.0
- Ratio: 9:16 / 16:9 / 1:1 / 3:4
- Language: zh-TW
- Status: Draft

## Purpose
建立符合品牌規範、資訊層級清楚且可用於招生、行銷、活動或企業溝通的海報。

## Inputs
- 主標題：
- 副標題：
- 核心訊息：
- 目標受眾：
- 行動呼籲：
- 必放 Logo／人物／產品：
- 禁止項目：

## Outputs
- 主視覺檔
- 可編輯來源檔
- 社群裁切版本
- 品質檢查報告

## Required Sections
1. 品牌識別區
2. 主標題區
3. 核心內容區
4. 證據或利益點區
5. 行動呼籲與必要資訊

## Optional Sections
- 講師或人物介紹
- QR Code
- 贊助與主辦單位
- 法規與異動聲明

## Validation Rules
- 標題在縮圖尺寸仍可辨識。
- 文字使用繁體中文且無錯字。
- 色彩、字級、間距符合 `../docs/03-color-system.md`、`../docs/04-typography.md`、`../docs/05-layout-grid.md`。
- Logo 不變形、不被裁切、不使用錯誤版本。
- 不新增未核准資訊。

## Example
```yaml
title: 企業級 AI Agent 架構
ratio: 9:16
audience: 企業主管與專案團隊
cta: 立即報名
style: 專業、科技、高質感
```
