# Infographic Template

## Metadata
- Title:
- Topic:
- Owner:
- Version: 1.0.0
- Ratio:
- Audience:

## Purpose
將複雜概念轉為可快速理解、比較與記憶的資訊圖表。

## Inputs
- 核心問題：
- 主要資料：
- 分類方式：
- 關鍵數字：
- 資料來源：

## Outputs
- 完整資訊圖
- 無文字背景版（選用）
- 資料與來源清單

## Required Sections
1. 標題與一句話結論
2. 3–7 個主要資訊群組
3. 圖例、流程或比較關係
4. 來源與版本

## Optional Sections
- 時間軸
- 象限圖
- 成熟度階梯
- KPI 卡片

## Validation Rules
- 每個區塊只表達一個主要觀點。
- 數字、單位與來源一致。
- 不以裝飾取代資訊。
- 圖示含義一致並符合 `../docs/07-icon-system.md`。

## Example
```yaml
structure: 五層成熟度階梯
message: 從工具使用走向企業治理
source: approved internal framework
```
