# AI Image Template

## Metadata
- Asset Name:
- Owner:
- Version: 1.0.0
- Model or Tool:
- Ratio:
- Language:

## Purpose
建立可重複、可追蹤且符合企業品牌規範的 AI 圖像生成提示詞。

## Inputs
- 圖像用途：
- 主體：
- 場景：
- 構圖：
- 品牌色與風格：
- 必放元素：
- 禁止元素：
- 參考素材：

## Outputs
- 完整提示詞
- 負面限制
- 生成參數
- 選圖與修訂紀錄

## Required Sections
1. Goal
2. Subject
3. Scene and Composition
4. Visual Language
5. Brand Constraints
6. Text Requirements
7. Negative Constraints
8. Output Specification

## Optional Sections
- 鏡頭與光線
- 材質
- 人物姿態
- 後製指示

## Validation Rules
- 使用正確 Logo、人物與產品版本。
- 繁體中文字需逐字校對。
- 不移除必要權利標示，不加入虛假背書。
- 遵循 `../docs/08-illustration-system.md` 與 `../docs/09-image-prompt-system.md`。

## Example
```yaml
ratio: 9:16
style: 專業、科技、高質感
subject: Codex 口袋魔法師
constraints: 不新增 QR code、不使用錯誤 Logo、繁體中文
```
